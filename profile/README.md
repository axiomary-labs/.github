# Axiomary Labs
### *The Sovereign Logic Verification Engine*

Axiomary is a high-performance, local-first logic mapping tool built in **Rust**. We bridge the gap between business requirements (Intent) and source code (Reality) without the risks of cloud-based AI or non-deterministic security audits.

---

## 🛡️ The Sovereign Mission
Current security tools find vulnerabilities, but they don't verify **Logic**. Axiomary uses **Tree-sitter** AST extraction and local inference to prove that your code does exactly what your documentation says it does.

* **100% Local:** No source code ever leaves your infrastructure.
* **Deterministic:** Logic mapping backed by Abstract Syntax Trees, not just LLM "guesses."
* **Enterprise Grade:** Optimized for Spring Boot (Java) and Go microservices.

## 🛠️ Tech Stack
- **Engine:** Rust
- **Parsing:** Tree-sitter
- **Graphing:** Petgraph
- **Intelligence:** Local GGUF Inference (Llama-3 / Phi-4)
