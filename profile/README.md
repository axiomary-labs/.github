## 🚀 The Mission
**Axiomary** is a high-performance, local-first logic mapping engine. We bridge the critical gap between **Business Requirements (Intent)** and **Source Code (Reality)** using deterministic AST analysis and sovereign AI.

While traditional tools (SAST/DAST) find vulnerabilities, they fail to verify if the code actually performs the business logic it was designed for. **We solve the Logic Gap.**

---

## 🏗️ Core Pillars

### 🧩 Deterministic Mapping
Powered by **Tree-sitter**, Axiomary extracts Abstract Syntax Trees (ASTs) to map code pathways directly to requirement nodes. No "hallucinations," only proven logic.

### 🏠 100% Sovereign
Designed for air-gapped enterprise environments. No source code ever leaves your infrastructure. Local-first inference ensures data privacy is never compromised.

### 🦀 Rust-Powered Performance
Built from the ground up in Rust for memory safety and the speed required to analyze million-line Enterprise Java (Spring Boot) and Go microservices.

---

## 🛠️ Tech Stack & Integration
- **Parser Engine:** [Tree-sitter](https://tree-sitter.github.io/tree-sitter/)
- **Graph Database:** [Petgraph](https://github.com/petgraph/petgraph) (In-memory)
- **Local LLM:** [Candle](https://github.com/huggingface/candle) / Llama.cpp (GGUF)
- **Languages Supported:** Java (Spring), Go, Rust.
